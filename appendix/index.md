# Appendix Index

## Appendix A (`appendix/appendix_a`)
- `rai_formula.md` — RAI formula and variable definitions for transmission density, HV line count, and proximity.
- `rai_sensitivity.py` — Script to run quartile sensitivity under three weighting schemes.
- `rai_sensitivity_table.csv` — County-level quartile assignment table plus stability summary row.
- `rai_sensitivity_table.md` — Markdown version of the RAI sensitivity table.
- `rai_sensitivity_chart.png` — Top-quartile overlap chart across weighting schemes.

## Appendix B (`appendix/appendix_b`)
- `cost_model_appendix.py` — Script for cost-model assumptions, SCC sensitivity, and chart generation.
- `cost_model_inputs.csv` — Input assumptions and derived 8,900 MW allocation table.
- `cost_model_inputs.md` — Markdown version of the input assumptions table.
- `cost_model_sensitivity.csv` — Baseline vs SCC sensitivity output table.
- `cost_model_sensitivity.md` — Markdown version of sensitivity table.
- `cost_sensitivity_chart.png` — Net-benefit sensitivity bar chart under two SCC assumptions.
- `cost_model_methodology.md` — Plain-English explanation of the 8,900 MW derivation and zone weighting.

## Appendix C (`appendix/appendix_c`)
- `pipeline_summary.py` — Script to build source/type/status pipeline summary and chart.
- `pipeline_summary.csv` — Pipeline table by source, facility type, status, count, and MW.
- `pipeline_summary.md` — Markdown version of the pipeline summary table.
- `pipeline_breakdown.png` — Stacked bar chart of facility type by committed vs planned capacity.
- `pipeline_notes.md` — Notes on DOE vs scraper fields, committed/planned distinction, and what is published on GitHub versus Oxford Economics proprietary scraper data.

## Appendix D (`appendix/appendix_d`)
- `build_stakeholder_matrix.py` — Script to generate stakeholder power-interest matrix image.
- `stakeholder_matrix.png` — Stakeholder matrix figure for appendix.
- `stakeholder_notes.md` — Four-sentence stakeholder interpretation notes.
